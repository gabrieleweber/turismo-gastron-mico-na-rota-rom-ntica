# turismo-gastron-mico-na-rota-rom-ntica

1.	Descrição do Projeto
O projeto consiste no desenvolvimento de uma aplicação Front-End para divulgação dos atrativos gastronômicos da Rota Romântica, localizada no Rio Grande do Sul.
A aplicação permitirá que os usuários visualizem restaurantes, cafés coloniais, vinícolas, cervejarias artesanais e eventos gastronômicos da região através de uma interface moderna, intuitiva e responsiva.
O desenvolvimento será realizado utilizando React e React Router, seguindo a estrutura de componentes e rotas apresentada no guia de wireframes fornecido.

2.	Objetivo Geral
Desenvolver uma aplicação web Front-End para promover o turismo gastronômico da Rota Romântica, oferecendo uma experiência de navegação agradável e organizada aos usuários.
